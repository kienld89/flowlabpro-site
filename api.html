Hướng Dẫn Sử Dụng API FlowLab Pro FlowLab Pro API Guide

Tài liệu hướng dẫn tích hợp, cấu hình và gọi các API của FlowLab Pro để tự động hóa quy trình tạo ảnh, video, lipsync, upscale và quản lý Assets. Documentation for integrating, configuring, and calling FlowLab Pro APIs to automate image, video, lipsync, upscale, and asset management.

🌐 Endpoint: http://127.0.0.1:1241
⚡ Chế độ: Đồng bộ (Synchronous) ⚡ Mode: Synchronous

1. Hướng Dẫn Kết Nối & Xác Thực 1. Connection & Authentication Guide

1.1 Khởi Chạy Kết Nối 1.1 Launching Connection

  1. Khởi chạy file flowlab-local-helper.exe (API Server sẽ chạy ngầm tại cổng 1241). Run the flowlab-local-helper.exe file (API Server will run in the background on port 1241).
  2. Mở trình duyệt Chrome và truy cập trang Google Labs. Open Chrome browser and navigate to Google Labs.
  3. Mở Sidebar UI của Extension FlowLab Pro. Open the Sidebar UI of the FlowLab Pro Extension.
  4. Chấm tròn thứ 4 (API Server) tại phần header của Sidebar chuyển sang màu xanh lá cây báo hiệu kết nối thành công, hoặc mở http://127.0.0.1:1241/health để kiểm tra. The 4th status dot (API Server) in the Sidebar header turns green to indicate success, or visit http://127.0.0.1:1241/health to check.

1.2 Headers Xác Thực Bắt Buộc 1.2 Required Authentication Headers

Mọi yêu cầu gửi tới REST API (ngoại trừ /health) đều yêu cầu đính kèm các header sau: All requests sent to the REST API (except /health) must include the following headers: 

Content-Type: application/json
Authorization: Bearer <YOUR_API_KEY>

2.0 Hướng Dẫn Tham Số Model Tùy Chọn 2.0 Optional Model Parameter Guide

Đối với tất cả các API tạo ảnh, video, lipsync, và asset (Character, Prop, Location), bạn có thể gửi kèm thuộc tính "model" (kiểu String, tùy chọn) trong request body để chỉ định mô hình sử dụng. Nếu không truyền, hệ thống sẽ sử dụng mô hình được thiết lập trên Extension Sidebar. For all image, video, lipsync, and asset APIs, you can optionally include the "model" parameter in the request body to force generation with a specific model. If omitted, the default model configured in the Extension Settings will be used.

Text To Image (Tạo ảnh từ văn bảnGenerate image from text)

POST /api/text-to-image

Tạo ảnh nghệ thuật mới dựa trên mô tả văn bản. Create a new artwork based on a text prompt.

Tham sốParameter KiểuType Yêu cầuRequirement Mô tảDescription
prompt String Bắt buộcRequired Mô tả chi tiết bức ảnh cần sinh ra. Detailed description of the image to generate.
aspectRatio String Bắt buộcRequired Tỉ lệ ảnh. Giá trị hợp lệ: "9:16", "16:9", "1:1", "4:3", "3:4". Image aspect ratio. Valid values: "9:16", "16:9", "1:1", "4:3", "3:4".
model String Tùy chọnOptional Mô hình tạo ảnh cụ thể ("gem_pix_2", "narwhal", "imagen4"). Specific image model ("gem_pix_2", "narwhal", "imagen4").

Ví dụ cURL:cURL Example:

curl -X POST http://127.0.0.1:1241/api/text-to-image \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer <YOUR_API_KEY>" \
  -d "{\"prompt\": \"A futuristic city in the style of cyberpunk\", \"aspectRatio\": \"16:9\", \"model\": \"gem_pix_2\"}"

model String Tùy chọnOptional Mô hình tạo ảnh cụ thể ("gem_pix_2", "narwhal", "imagen4"). Specific image model ("gem_pix_2", "narwhal", "imagen4").

5. Xử Lý Lỗi (Error Handling) 5. Error Handling

Các HTTP status trả về khi có sự cố: HTTP status codes returned when issues occur:

  • 401 Unauthorized: Sai API Key hoặc phiên làm việc Google Labs hết hạn. Invalid API Key or Google Labs session expired.
  • 400 Bad Request: Thiếu tham số bắt buộc hoặc định dạng không hợp lệ. Missing required parameters or invalid format.
  • 429 Too Many Requests: Đạt giới hạn hạn mức (Rate limit) của tài khoản Google. Rate limit reached for the Google account.
  • 503 Service Unavailable: Extension trong Chrome chưa được mở hoặc không kết nối Helper. Chrome Extension is not running or not connected to the Helper.
  • 504 Gateway Timeout: Quá thời gian xử lý tác vụ sinh media. Timeout processing the media generation task.

6.1 Python (Gọi Text-to-Video API) 6.1 Python (Call Text-to-Video API) 

import requests

def generate_video(prompt, api_key):
    url = "http://127.0.0.1:1241/api/text-to-video"
    headers = {
        "Content-Type": "application/json",
        "Authorization": f"Bearer {api_key}"
    }
    payload = {
        "prompt": prompt,
        "aspectRatio": "16:9",
        "duration": "8s",
        "model": "fast"  # Truyền model tùy chọn (ví dụ: fast, quality, lite, lower)Pass optional model (e.g., fast, quality, lite, lower)
    }
    
    print("Đang tạo video... (Có thể mất 1-3 phút)")
    try:
        response = requests.post(url, json=payload, headers=headers, timeout=240)
        
        if response.status_code == 200:
            res_data = response.json()
            if res_data.get("success"):
                print("Tạo thành công!")
                print("Download Link:", res_data["result"]["url"])
                print("File vật lý đã tải về:", res_data["result"]["localPath"])
                return res_data["result"]
            else:
                print("Lỗi từ API:", res_data.get("error"))
        else:
            print(f"Lỗi hệ thống ({response.status_code}):", response.text)
            
    except requests.exceptions.Timeout:
        print("Lỗi: Quá thời gian chờ server phản hồi (Timeout).")
    except Exception as e:
        print("Lỗi kết nối:", str(e))
        
    return None

using System;
using System.Net.Http;
using System.Text;
using System.Text.Json;
using System.Threading.Tasks;

public class FlowLabClient
{
    private static readonly HttpClient client = new HttpClient { Timeout = TimeSpan.FromSeconds(240) };

    public static async Task GenerateImageToVideoAsync(string prompt, string imageUrl, string apiKey)
    {
        client.DefaultRequestHeaders.Clear();
        client.DefaultRequestHeaders.Add("Authorization", $"Bearer {apiKey}");

        var payload = new
        {
            prompt = prompt,
            mediaId = imageUrl,
            aspectRatio = "9:16",
            duration = "8s",
            model = "quality"  // Truyền model tùy chọn (ví dụ: fast, quality, lite, lower)Pass optional model (e.g., fast, quality, lite, lower)
        };

        var json = JsonSerializer.Serialize(payload);
        var content = new StringContent(json, Encoding.UTF8, "application/json");

        Console.WriteLine("Đang gửi yêu cầu tạo video...");
        try
        {
            var response = await client.PostAsync("http://127.0.0.1:1241/api/image-to-video", content);
            var resultString = await response.Content.ReadAsStringAsync();
            
            if (response.IsSuccessStatusCode)
            {
                Console.WriteLine("Thành công:");
                Console.WriteLine(resultString);
            }
            else
            {
                Console.WriteLine($"Lỗi API ({response.StatusCode}): {resultString}");
            }
        }
        catch (TaskCanceledException)
        {
            Console.WriteLine("Lỗi: Quá thời gian chờ server phản hồi (Timeout).");
        }
        catch (Exception ex)
        {
            Console.WriteLine("Lỗi ngoại lệ: " + ex.Message);
        }
    }
}
